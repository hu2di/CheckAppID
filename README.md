## Download
JitPack:
Step 1. Add the JitPack repository to your build file
Add it in your root build.gradle at the end of repositories:
```groovy
repositories {
    jcenter()
}
```
Step 2. Add the dependency
```groovy
dependencies {
    compile 'com.github.boxme:squarecamera:1.1.0'
}
```
